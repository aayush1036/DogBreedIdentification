Introduction

We see that many dogs are abandoned by the owners who couldn't take care of their dogs due to some reason, and abandon them, and the abandoned dogs are forced to live on the streets facing hunger, cold and unfriendly locals who sometimes get scared and hurt them. Being dog lovers, we aim to solve this problem with our project a--calypse which uses deep learning to identify dog breeds and alert the NGO so that they can rescue the dog and the dog can find a new home

Know Your Data

The dataset used in this project is sourced from Kaggle which is the Dog Breed Identification dataset Since the data did not have a dedicated test set with labels, we had to do a split while performing data augmentation where we had 80% of the data in the train set and 10% of the data in the validation set and 10% of the data in test set. We also performed certain data augmentations on the dataset which are:
  1. Resizing - We have resized all the images to 256,256 pixels so as to not diminish the image size which may result in loss of feautures and clarity and not big enough to increase the number of parameters in our model. This size is also the input size for many pre trained models which eases the training of models.
  2. Rescaling - We rescaled the pixels of the images to be in between 0 and 1 so that our model can converge faster
  3. Shearing - We performed shearing on the dataset to accomodate the shaking of the device which may have occured while clicking the photos. We kept the shearing co efficient to a low value of 0.2 so as to not distort the training data in such a way that the model cannot learn the features properly
  4. Zoom - We implemented the zooming augmentation to accomodate the varying distances from which the user can take photos, we have set the zoom range from 0.8x to 1.2x so as to not hamper the training data but adjust for slightly varying distances while taking the photo of the dog
  5. Horizontal flip - We have implemented the horizontal flip augmentation as the user may take the picture from the front or behind where the left and right sides may get flipped and to accomodate that we have implemented the horizontal flip data augmentation
